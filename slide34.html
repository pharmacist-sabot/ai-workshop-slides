Main Project: Step 1.3

ติดตั้งโค้ดหน้าบ้าน (HTML)

เราจะนำโค้ดฟอร์มที่ AI สร้างให้ มาสร้างเป็นไฟล์เว็บจริงในโปรเจกต์ของเรา

กลับมาที่หน้าต่าง Apps Script Editor แล้วทำตามขั้นตอนต่อไปนี้:

  1. ข้างๆ "Files" (ไฟล์), กดที่เครื่องหมาย + แล้วเลือก HTML
  2. ตั้งชื่อไฟล์ว่า Index (ตัว I พิมพ์ใหญ่, ไม่ต้องใส่ .html) แล้วกด Enter
  3. ในไฟล์ Index.html ที่เปิดขึ้นมา ให้ลบโค้ดเก่าทิ้งทั้งหมด
  4. วางโค้ด HTML ใหม่ ที่เราได้มาจาก AI ลงไปแทนที่
  5. กดที่ไอคอนรูปแผ่นดิสก์ เพื่อ Save Project
