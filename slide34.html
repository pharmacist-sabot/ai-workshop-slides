ติดตั้งโค้ดหน้าบ้าน (HTML)

เราจะนำโค้ดฟอร์มที่ AI สร้างให้ มาสร้างเป็นไฟล์เว็บจริงในโปรเจกต์ Apps Script ของเรา

ขั้นตอนการติดตั้ง:

  1. ที่หน้า Apps Script Editor, กด ข้างๆ "Files" (ไฟล์) แล้วเลือก HTML
  2. ตั้งชื่อไฟล์ว่า Index แล้วกด Enter
  3. ลบโค้ดเก่า ที่มีอยู่ 2 บรรทัดทิ้งให้หมด
  4. วางโค้ด HTML ใหม่ ที่ได้จาก AI ลงไปแทน
  5. กด Save project เพื่อบันทึก
