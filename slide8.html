3 Steps to Get Ready

เช็คลิสต์เตรียมความพร้อม

เราต้องการเพียง 3 สิ่งนี้เท่านั้น เพื่อเริ่มต้นการสร้าง Web App ของเรา

1

Google Account

สำหรับเข้าใช้งาน Google Sheets และ Apps Script (เชื่อว่าทุกคนมีพร้อม)

2

คอมพิวเตอร์ & อินเทอร์เน็ต

Workshop ของเราทำบนเว็บทั้งหมด ไม่ต้องติดตั้งโปรแกรมใดๆ เพิ่มเติม

3

Gemini API Key

กุญแจสำคัญสำหรับเชื่อมต่อกับ AI (สไลด์ถัดไปจะสอนวิธีรับกุญแจนี้)

ย้อนกลับ ไปรับ API Key กัน
