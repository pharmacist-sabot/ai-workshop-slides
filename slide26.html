สั่งงาน AI ด้วย Master Prompt

Copy Prompt ทั้งหมดนี้ ไปวางใน Gemini (หรือ AI ตัวอื่น) แล้วกด Enter

[บทบาท]คุณคือโปรแกรมเมอร์ผู้เชี่ยวชาญด้าน Google Apps Script และมีความรู้ด้านเภสัชกรรม [บริบท]ฉันกำลังสร้างเครื่องมือใน Google Sheet ที่ชื่อชีทว่า "Drug Summarizer" โดยคอลัมน์ A จะเป็น Drug Name และคอลัมน์ B จะเป็น AI Summary [งาน]โปรดเขียนโค้ดที่ทำงานดังนี้: 1. สร้างเมนูชื่อ "AI Tools" และเมนูย่อย "Summarize Drug Info" 2. เมื่อคลิกเมนูย่อย ให้ดึงชื่อยาจากเซลล์ A2 3. สร้าง Prompt ใหม่เพื่อขอข้อมูล "counseling points" ของยานั้นๆ เป็นภาษาไทยไม่เกิน 5 ข้อ 4. เรียกใช้ Gemini API ด้วย Prompt ใหม่ 5. นำผลลัพธ์ที่ได้ไปใส่ในเซลล์ B2 6. สร้างฟังก์ชัน `callGeminiAPI` ที่มีตัวแปร `API_KEY` ให้เติมค่าเอง [รูปแบบ]ขอเฉพาะโค้ด Apps Script ที่พร้อมใช้งานเท่านั้น
