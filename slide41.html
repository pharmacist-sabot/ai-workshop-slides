  4. กด Save Project

ตอนที่ 2: Deploy Version ใหม่

  1. กดปุ่มสีน้ำเงิน Deploy
  2. เลือก Manage deployments
  3. คลิกไอคอนรูปดินสอ เพื่อแก้ไข
  4. ในช่อง Version, เลือก New version (สำคัญมาก!)
  5. กดปุ่ม Deploy เพื่ออัปเดต Web App ของเรา
