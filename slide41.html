ติดตั้ง "สายไฟ" และ Deploy ครั้งสุดท้าย

ขั้นตอนสุดท้ายแล้ว เราจะนำทุกชิ้นส่วนมาประกอบกันและเปิดใช้งานระบบทั้งหมด

1ติดตั้ง JavaScript

  1. เปิดไฟล์ Index.html
  2. เลื่อนไปที่ล่างสุดของไฟล์
  3. วางโค้ด <script> ก่อน </body>
  4. กด Save project

2Deploy Version ใหม่

  1. กด Deploy > Manage
  2. คลิกไอคอนดินสอ (Edit)
  3. เลือก Version เป็น "New version"
  4. กด Deploy เพื่ออัปเดต
