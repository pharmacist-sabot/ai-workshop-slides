Main Project: Step 2.2

ทดสอบครั้งที่ 1: นำเว็บขึ้นออนไลน์ (Deploy)!

เราจะ "ปล่อยของ" เปลี่ยนโค้ดของเราให้กลายเป็น Web App ที่มีชีวิตและมี URL จริง

ทำตามขั้นตอนการ Deploy อย่างระมัดระวังนะครับ:

  1. ที่มุมบนขวาของ Apps Script Editor, กดปุ่มสีน้ำเงิน Deploy และเลือก New deployment
  2. คลิกไอคอนรูปเฟือง และเลือก Type เป็น Web app
  3. ในส่วน Who has access, ให้เลือกเป็น Anyone (เพื่อให้ทุกคนสามารถเข้าทดสอบได้โดยไม่ต้องล็อกอิน)
  4. กดปุ่ม Deploy (อาจมีการขอ Authorization อีกครั้ง ให้กด Allow)
  5. เมื่อสำเร็จ, Copy URL ของ Web app ที่ได้มา แล้วนำไปเปิดใน Tab ใหม่ของเบราว์เซอร์
ย้อนกลับ ไปดูผลลัพธ์การ Deploy
