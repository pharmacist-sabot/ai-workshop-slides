Deployment Guide

  1. กดปุ่มสีน้ำเงิน Deploy และเลือก New deployment
  2. คลิก และเลือก Type เป็น Web app
  3. ในส่วน Who has access,
    ให้เลือกเป็น "Anyone"
  4. กดปุ่ม Deploy (และ "Allow" หากมีการขออนุญาต)
  5. Copy Web app URL ที่ได้ แล้วนำไปเปิดในแท็บใหม่
