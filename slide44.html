สร้าง Backend สำหรับดึงข้อมูล

เราจะสร้างโปรเจกต์ใหม่สำหรับ Dashboard และสั่ง AI ให้เขียนโค้ดดึงข้อมูลจาก Sheet เดิมของเรา

ขั้นตอนเตรียมการ:

  1. สร้างโปรเจกต์ Apps Script ใหม่
    (ไปที่ Google Drive > + New)
  2. ตั้งชื่อโปรเจกต์ว่า Risk Dashboard
  3. Copy Spreadsheet ID จาก URL ของไฟล์ Sheet "AI Pharmacist Tools"

สั่งงาน AI

นำ Sheet ID ที่คัดลอกมา
ไปใส่ใน Prompt ที่กำลังจะแสดงนี้

ย้อนกลับ ต่อไป: สร้างหน้า Dashboard

Backend Master Prompt

คุณคือโปรแกรมเมอร์ Google Apps Script โปรดเขียนฟังก์ชันชื่อ `getDataForDashboard()` โดยให้ทำงานดังนี้: 1. เข้าถึง Spreadsheet ด้วย ID โดยใช้ `SpreadsheetApp.openById("YOUR_SHEET_ID_HERE")` **<-- วาง ID ของคุณที่นี่** 2. เลือกชีทที่ชื่อว่า `Risk_Log` 3. อ่านข้อมูลทั้งหมดในชีท (ใช้ `getDataRange().getValues()`) 4. ดึงแถว Header (แถวแรกสุด) ออกมาเก็บไว้ 5. วนลูปข้อมูลที่เหลือ (ไม่รวม Header) เพื่อแปลงข้อมูลจาก Array 2 มิติ ให้อยู่ในรูปแบบ Array ของ Objects โดยให้แต่ละ Object มี key ตรงกับชื่อ Header ที่ดึงมาในข้อ 4 6. Return Array ของ Objects นี้ออกมา ขอเฉพาะโค้ดฟังก์ชัน `getDataForDashboard` เท่านั้น
