สั่ง AI เขียน "สายไฟ" (JavaScript)

เราจะสร้างโค้ด JavaScript เพื่อเชื่อมปุ่ม Submit ในฟอร์มของเรา ให้ส่งข้อมูลไปหาฟังก์ชัน saveData

หัวใจของ Prompt นี้:

  • บอก AI ว่าเราใช้ google.script.run ซึ่งเป็นคำสั่งพิเศษของ Apps Script
  • สั่งงานเป็นขั้นตอนที่โปรแกรมเมอร์เข้าใจ (ดัก event, ป้องกัน reload, เรียกฟังก์ชันหลังบ้าน)
  • ระบุคำสั่งสำคัญ เช่น .withSuccessHandler() และ .withFailureHandler()
ย้อนกลับ ต่อไป: ติดตั้ง JavaScript

JavaScript Master Prompt

คุณคือโปรแกรมเมอร์ JavaScript ผู้เชี่ยวชาญ ฉันมีฟอร์ม HTML ที่มีปุ่ม submit อยู่ภายใน <form> tag และมีฟังก์ชันหลังบ้านของ Google Apps Script ชื่อ `saveData(formObject)` ที่พร้อมรับข้อมูลจากฟอร์ม โปรดเขียนโค้ด JavaScript ทั้งหมดที่ต้องใส่ใน <script> tag สำหรับไฟล์ `Index.html` โดยโค้ดจะต้อง: 1. หา form element ในหน้าเว็บ 2. ดักฟัง (listen for) event 'submit' ของ form นั้น 3. เมื่อ form ถูก submit, ให้สั่ง `event.preventDefault()` เพื่อป้องกันไม่ให้หน้าเว็บโหลดใหม่ 4. เรียกใช้ฟังก์ชันหลังบ้าน `saveData` ผ่าน `google.script.run` 5. ใส่ `.withSuccessHandler()` เพื่อจัดการเมื่อโค้ดหลังบ้านทำงานสำเร็จ โดยให้แสดง alert box ด้วยข้อความที่ส่งกลับมา แล้วทำการ reset ฟอร์มให้ว่าง 6. ใส่ `.withFailureHandler()` เพื่อจัดการเมื่อเกิดข้อผิดพลาด โดยให้แสดง alert box พร้อมข้อความ error 7. ส่ง `event.target` (ซึ่งก็คือ form object) เป็น argument เข้าไปในฟังก์ชัน `saveData` ขอเฉพาะโค้ดที่อยู่ภายใน <script> และ </script> เท่านั้น
