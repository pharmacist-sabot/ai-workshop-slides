Main Project: Step 2.1

สั่ง AI เขียนโค้ด "เปิดประตูเว็บ"

เราต้องสร้างฟังก์ชันพิเศษชื่อ `doGet` ซึ่งทำหน้าที่เป็นทางเข้าหลักของ Web App

1. Copy Prompt นี้ไปสั่ง AI:

บทบาท: โปรแกรมเมอร์ Google Apps Script งาน: เขียนฟังก์ชัน `doGet` เพื่อแสดงผล (serve) ไฟล์ HTML ที่ชื่อว่า `Index.html` ให้เป็น Web App

2. นำโค้ดที่ได้ไปติดตั้ง:

  1. กลับไปที่ Apps Script Editor
  2. เปิดไฟล์ Code.gs
  3. ลบฟังก์ชัน myFunction ที่มีอยู่เดิมทิ้ง
  4. วางโค้ด `doGet` ที่ได้จาก AI ลงไป
  5. กด Save Project

เมื่อทำเสร็จแล้ว เราก็พร้อมที่จะ "Deploy" หรือนำเว็บของเราขึ้นออนไลน์เป็นครั้งแรกในสไลด์ถัดไป!

ย้อนกลับ ต่อไป: Deploy!
