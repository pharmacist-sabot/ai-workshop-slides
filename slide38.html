Main Project: Step 2.3

สั่ง AI เขียนโค้ด "รับข้อมูล" (Backend Logic)

เราจะสร้างฟังก์ชัน `saveData` ที่ทำหน้าที่เป็น "เจ้าหน้าที่" รับข้อมูลจากฟอร์มไปบันทึกลง Sheet

บทบาท: โปรแกรมเมอร์ Google Apps Script ผู้เชี่ยวชาญ บริบท: ฉันมี Web App ที่สร้างจากไฟล์ HTML ชื่อ `Index.html` และมี Google Sheet ชื่อ `Risk_Log` ที่มีคอลัมน์ตามลำดับดังนี้: Timestamp, ReporterEmail, EventDate, PatientID, EventDescription, MedicationInvolved, RiskCategory. งาน: โปรดเขียนฟังก์ชันชื่อ `saveData(formObject)` โดยให้ทำงานดังนี้: 1. เข้าถึง Spreadsheet ที่กำลังใช้งานอยู่ และเลือกชีทที่ชื่อว่า `Risk_Log` 2. ดึงอีเมลของผู้ใช้ที่กำลังล็อกอินอยู่ (Session.getActiveUser().getEmail()) 3. เพิ่มแถวใหม่ (appendRow) ต่อท้ายข้อมูลที่มีอยู่ 4. ข้อมูลในแถวใหม่จะต้องเรียงตามลำดับคอลัมน์ของชีท คือ: - วันที่และเวลาปัจจุบัน (new Date()) - อีเมลของผู้ใช้ที่ดึงมา - ข้อมูลจาก `formObject.eventDate` - ข้อมูลจาก `formObject.patientId` - ข้อมูลจาก `formObject.eventDescription` - ข้อมูลจาก `formObject.medicationInvolved` - ข้อมูลจาก `formObject.riskCategory` 5. เมื่อบันทึกสำเร็จ ให้ return ข้อความกลับไปว่า "บันทึกรายงานสำเร็จ!" รูปแบบ: ขอเฉพาะโค้ดฟังก์ชัน `saveData` เท่านั้น
