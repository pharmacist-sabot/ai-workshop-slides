สั่ง AI สร้าง "หลังบ้าน" (Backend)

เราจะสร้างฟังก์ชัน saveData ที่ทำหน้าที่เป็น "เจ้าหน้าที่" รับข้อมูลจากฟอร์มไปบันทึกลง Sheet

หัวใจของ Prompt นี้:

  • Context: บอกชื่อชีท (Risk_Log) และลำดับคอลัมน์
  • Task: สั่งงานเป็นขั้นตอนชัดเจน (เพิ่มแถว, เรียงข้อมูล)
  • Specifics: ระบุชื่อฟังก์ชัน (saveData) และพารามิเตอร์
  • Format: ขอเฉพาะโค้ดฟังก์ชันเดียว เพื่อให้กระชับ
ย้อนกลับ ต่อไป: ติดตั้งโค้ด Backend

Backend Master Prompt

คุณคือโปรแกรมเมอร์ Google Apps Script ผู้เชี่ยวชาญ ฉันมี Google Sheet ชื่อ "Risk_Log" ที่มีคอลัมน์ตามลำดับดังนี้: Timestamp, EventDate, PatientType, ErrorCategory, Description, SeverityLevel โปรดเขียนฟังก์ชันชื่อ `saveData(formObject)` โดยให้ทำงานดังนี้: 1. เข้าถึงชีทที่ชื่อว่า "Risk_Log" 2. เพิ่มแถวใหม่ (appendRow) ต่อท้ายข้อมูลที่มีอยู่ 3. ข้อมูลในแถวใหม่จะต้องเรียงตามลำดับคอลัมน์ของชีท คือ: - วันที่และเวลาปัจจุบัน (new Date()) - ข้อมูลจาก `formObject.EventDate` - ข้อมูลจาก `formObject.PatientType` - ข้อมูลจาก `formObject.ErrorCategory` - ข้อมูลจาก `formObject.Description` - ข้อมูลจาก `formObject.SeverityLevel` 4. เมื่อบันทึกสำเร็จ ให้ return ข้อความกลับไปว่า "บันทึกรายงานสำเร็จ!" ขอเฉพาะโค้ดฟังก์ชัน `saveData` เท่านั้น
