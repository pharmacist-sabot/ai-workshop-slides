เปิดประตู Dashboard และ Deploy

ขั้นตอนสุดท้าย เราจะนำทุกชิ้นส่วนมาประกอบกันและนำ Dashboard ของเราขึ้นออนไลน์

1. ตรวจสอบ function doGet

ตรวจสอบฟังก์ชัน doGet() เพื่อบอกให้ Apps Script รู้ว่าต้องแสดงไฟล์ HTML ไหนเมื่อมีคนเข้ามาที่ลิงก์

2. Deploy

เผยแพร่โปรเจกต์เป็น Web App และตั้งค่าการเข้าถึงเป็น "Anyone" เพื่อให้ทุกคนสามารถเข้ามาดู Dashboard ได้

3. Launch

คัดลอก URL ที่ได้จากการ Deploy นี่คือลิงก์ Dashboard ที่พร้อมใช้งานและแชร์ให้คนอื่นได้ทันที
