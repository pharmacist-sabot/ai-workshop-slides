Deployment Guide

  1. เปิดไฟล์ Code.gs แล้วเพิ่มฟังก์ชัน doGet นี้เข้าไป: 
    function doGet() {
  return HtmlService
    .createHtmlOutputFromFile('Dashboard');
}
  2. กด Save Project
  3. Deploy โปรเจกต์ใหม่นี้ (Deploy > New deployment > Web app > Access: Anyone)
  4. Copy URL ใหม่ ที่ได้มา นี่คือลิงก์ Dashboard ของคุณ!
