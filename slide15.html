R-C-T-F Breakdown

"C" is for Context (บริบท)

ให้ข้อมูลเบื้องหลังที่จำเป็น เพื่อให้ AI ไม่ต้อง "เดา" และให้คำตอบได้ตรงประเด็น

แบบทั่วไป (ไม่มีบริบท)

"Act as a Google Apps Script expert. Write a function to check for errors."

ผลลัพธ์ที่อาจได้:
AI อาจจะสร้างฟังก์ชันกว้างๆ ที่ใช้ตรวจสอบ error ทั่วไปในโค้ด ซึ่งไม่เกี่ยวกับงานของเราเลย

(ทำงานได้ แต่ไม่ตรงกับที่เราต้องการ)

แบบให้บริบท

"Act as a Google Apps Script expert. I am a pharmacist building a web app to reduce medication errors. The app has a textarea for prescription input and a button. Write a function to check for errors."

ผลลัพธ์ที่ดีขึ้น:
AI จะเข้าใจทันทีว่า "errors" ในที่นี้หมายถึง "medication errors" และอาจจะสร้างฟังก์ชันชื่อ `checkForMedicationErrors(prescriptionText)` ที่พร้อมรับข้อมูลจาก textarea มาวิเคราะห์ต่อ

(ตรงเป้าหมาย และช่วยให้เราทำงานต่อได้เร็วขึ้นมาก)
