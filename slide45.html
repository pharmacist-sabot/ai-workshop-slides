Dashboard Master Prompt

คุณคือ Web Developer ผู้เชี่ยวชาญด้าน Data Visualization โปรดสร้างโค้ด HTML ทั้งหมดสำหรับไฟล์ชื่อ 'Dashboard.html' โดยมีเงื่อนไขดังนี้: 1. ใช้ Bootstrap 5 และ Chart.js version 3 จาก CDN 2. มีหัวข้อหลักว่า "Medication Risk Dashboard" 3. มีส่วนแสดงผลตัวเลขสรุป (Card) 1 อันสำหรับ "จำนวนรายงานทั้งหมด" โดยให้มี id คือ 'totalReportsCard' 4. มี <canvas> สำหรับวาดกราฟวงกลม (Pie Chart) 1 อัน โดยให้มี id คือ 'riskCategoryChart' 5. มีตาราง (Table) สำหรับแสดง "5 รายงานล่าสุด" โดยให้ <tbody> ของตารางมี id คือ 'latestReportsTableBody' 6. เขียนโค้ด JavaScript ทั้งหมดใน <script> tag โดยให้: a. เมื่อหน้าเว็บโหลด (DOMContentLoaded), ให้แสดงข้อความ Loading และเรียกใช้ฟังก์ชันหลังบ้าน 'getDataForDashboard()' ผ่าน `google.script.run` b. สร้าง Success Handler 'handleData(data)' ที่จะทำงานเมื่อได้รับข้อมูลกลับมา c. ในฟังก์ชัน 'handleData(data)', ให้ประมวลผลข้อมูลเพื่อ: - นับจำนวนรายงานทั้งหมด (data.length) แล้วแสดงผลใน '#totalReportsCard' - นับจำนวนของแต่ละ 'ErrorCategory' เพื่อเตรียมข้อมูลสำหรับกราฟ แล้วใช้ Chart.js วาด Pie Chart ลงบน '#riskCategoryChart' - นำข้อมูล 5 แถวสุดท้าย (ข้อมูลล่าสุด) มาสร้างเป็นแถว '<tr>' ในตาราง '#latestReportsTableBody' ขอโค้ด HTML ที่สมบูรณ์พร้อม JavaScript ภายในไฟล์เดียว
