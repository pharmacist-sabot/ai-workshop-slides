สั่ง AI สร้างหน้าฟอร์ม (HTML)

เราจะใช้ Prompt ที่สมบูรณ์ สั่งให้ AI เขียนโค้ดหน้าบ้าน (Frontend) ทั้งหมดให้เรา

หัวใจของ Prompt นี้:

  • Role: สั่งให้เป็นผู้เชี่ยวชาญ Bootstrap 5
  • Context: บอกว่าจะนำไปใช้ใน Google Apps Script
  • Task: ระบุชนิดของ Input แต่ละช่องอย่างชัดเจน
  • Format: ขอเฉพาะโค้ด HTML ที่สมบูรณ์เท่านั้น
ย้อนกลับ ต่อไป: ติดตั้งโค้ด HTML

HTML Master Prompt

คุณคือ Senior Frontend Developer ที่เชี่ยวชาญด้าน HTML และ Bootstrap 5 ฉันกำลังสร้าง Web App ที่จะรันบน Google Apps Script สำหรับให้เภสัชกรใช้บันทึกความเสี่ยงด้านยา โปรดสร้างโค้ดสำหรับไฟล์ HTML ไฟล์เดียว ที่ทำงานตามเงื่อนไขต่อไปนี้ทั้งหมด: 1. ใช้ Bootstrap 5 CDN ล่าสุด (CSS และ JS) 2. ออกแบบ Layout ให้ดูสะอาดตา ทันสมัย และ Responsive 3. ใส่หัวข้อหลักว่า "Medication Safety Reporting Form" 4. สร้างฟอร์มที่มีช่องกรอกข้อมูลต่อไปนี้ พร้อม label ที่ชัดเจน: - วันที่เกิดเหตุ (EventDate): ใช้ <input type="date"> - ประเภทผู้ป่วย (PatientType): ใช้ <select> ที่มีตัวเลือก "ผู้ป่วยนอก" และ "ผู้ป่วยใน" - ระดับความรุนแรง (SeverityLevel): ใช้ <select> ที่มีตัวเลือก "A" ถึง "I" - รายละเอียดเหตุการณ์ (Description): ใช้ <textarea> ที่มี 4 แถว 5. มีปุ่ม Submit สีน้ำเงิน (primary) ที่มีข้อความว่า "Submit Report" ขอเฉพาะโค้ด HTML ที่สมบูรณ์และพร้อมใช้งานเท่านั้น ไม่ต้องมีคำอธิบายอื่น
