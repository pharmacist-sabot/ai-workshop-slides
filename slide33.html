Main Project: Step 1.2

สั่ง AI สร้างหน้าฟอร์ม (The HTML Prompt)

เราจะใช้ Prompt ภาษาไทย สั่งให้ AI เขียนโค้ด HTML ที่มีหน้าตาสวยงามด้วย Bootstrap 5

บทบาท: คุณคือ Web Developer ผู้เชี่ยวชาญ งาน: โปรดสร้างโค้ด HTML ทั้งหมดสำหรับหน้าเว็บฟอร์ม "บันทึกความเสี่ยงด้านยา" โดยมีเงื่อนไขดังนี้: 1. ใช้ Bootstrap 5 จาก CDN เพื่อความสวยงาม (ไม่ต้องมีโค้ด CSS เอง) 2. ใส่หัวข้อหลักว่า "Medication Safety / Risk Reporting Form" 3. ฟอร์มจะต้องมีช่องกรอกข้อมูลต่อไปนี้ พร้อม `label` ที่ชัดเจน: - ช่องรับข้อมูล "วันที่เกิดเหตุ" (Event Date) โดยใช้ input type="date" - ช่องรับข้อมูล "รหัสผู้ป่วย (HN/AN)" (Patient ID) โดยใช้ input type="text" - ช่องรับข้อมูล "ยาที่เกี่ยวข้อง" (Medications Involved) โดยใช้ input type="text" - ช่องรับข้อมูล "ประเภทความเสี่ยง" (Risk Category) โดยใช้ dropdown (select) ซึ่งมีตัวเลือกคือ: 'Prescribing Error', 'Dispensing Error', 'Administration Error', 'Monitoring Error' - ช่องรับข้อมูล "รายละเอียดเหตุการณ์" (Event Description) โดยใช้ textarea ที่มีขนาด 5 แถว 4. ทุกช่องต้องมี `name` attribute ที่สอดคล้องกับ label (เช่น `name="eventDate"`) 5. มีปุ่ม Submit สีน้ำเงิน (primary) ที่มีข้อความว่า "Submit Report" รูปแบบ: ขอเฉพาะโค้ด HTML ที่สมบูรณ์เท่านั้น ไม่ต้องมีคำอธิบายอื่น
