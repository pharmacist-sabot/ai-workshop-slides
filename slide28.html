ไขรหัสเบื้องหลัง: API คืออะไร?

API Key และ API Endpoint ที่เราเพิ่งใช้ไป... มันคืออะไรกันแน่?

API Endpoint

เปรียบเสมือน: ที่อยู่ของโรงแรม (เช่น Google Hotel)

  • คือ "ที่อยู่" บนอินเทอร์เน็ตที่เราต้องส่งคำขอไปหา AI
  • เป็นที่อยู่สาธารณะ ใครๆ ก็รู้ได้
  • บอกว่าเรากำลังจะไปใช้บริการของใคร (ในที่นี้คือ Gemini)

API Key

เปรียบเสมือน: คีย์การ์ดเข้าห้องพักของคุณ

  • คือ "รหัสลับ" ที่พิสูจน์ว่าเราเป็นใคร และมีสิทธิ์เข้าใช้บริการ
  • เป็นของส่วนตัว ห้ามให้ใครเห็นเด็ดขาด!
  • ใช้ยืนยันตัวตนและนับโควต้าการใช้งานของเรา
ย้อนกลับ ไปทดสอบผลลัพธ์กัน
