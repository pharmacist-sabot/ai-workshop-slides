Quick Win: Step 4/4 (Installation)

ติดตั้งและตั้งค่าโค้ด

มาติดตั้งโค้ดที่ AI สร้างให้ลงใน "โรงงาน" (Apps Script) ของเรากันครับ

  1. กลับไปที่หน้าต่าง Apps Script ที่เราเปิดค้างไว้
  2. ลบโค้ดเก่า (`function myFunction...`) ที่มีอยู่ทิ้งให้หมด
  3. วางโค้ดใหม่ ที่ได้จาก AI ลงไปแทนที่
  4. (สำคัญที่สุด!) หาบรรทัด const API_KEY = "..." แล้วนำ API Key ที่เราเก็บไว้มาวางแทนที่ "YOUR_API_KEY_HERE" (ต้องมีเครื่องหมายคำพูดครอบไว้เหมือนเดิม)
  5. กดที่ไอคอนรูปแผ่นดิสก์ เพื่อ Save Project
ย้อนกลับ ไปดูผลลัพธ์กัน!
